
SDSS filter curves
------------------
Source: http://www.sdss.org/instruments/camera/ , original "Gunn" curves
Reference:
Filenames: SDSS-*.dat  (*= u, g, r, i, z)
Comment: Throughput is measured at airmass 1.3 for a point source
Units: lambda/Angstroem 	 throughput/au

HSC filter curves
-----------------
Source: https://hsc-release.mtk.nao.ac.jp/doc/index.php/survey/
Reference: http://adsabs.harvard.edu/abs/2018PASJ...70...66K
Filenames: HSC-*.dat (*= g, r, i, z, y)
Comment: For the atmosphere, we assumed airmass=1.2 (which is close to the median airmass of the survey) and PWV=1.5mm
Units: lambda/Angstroem  RSR/au


LSST filter curves
------------------
Source: https://github.com/lsst/throughputs/tree/master/baseline/ "total"
Reference:
Filenames: LSST-*.dat (*= u, g, r, i , z, y)
Comment: LSST filter responses are taken from tag 12.0 of the LSST simulations throughputs package and include a standard atmosphere with airmass 1.2 that is also tabulated in the same package
Units: wavelength/nm  RSR/au
